# Aturan untuk DeepSeek Coding Agent

## Bahasa
Selalu jawab dan beri penjelasan dalam Bahasa Indonesia, kecuali kode program (tetap dalam bahasa pemrograman yang sesuai).

## Scope Kerja
- Hanya buat, baca, atau ubah file di dalam folder proyek yang sedang aktif.
- Jangan akses file di luar direktori kerja tanpa izin eksplisit.
- Sebelum mengubah file yang sudah ada, tampilkan dulu rencana perubahan.

## Gaya Coding
- Tulis kode yang bersih, terdokumentasi, dan mudah dibaca.
- Tambahkan komentar singkat untuk logika yang tidak jelas.
- Ikuti konvensi bahasa yang digunakan dalam proyek.

## Keamanan
- Jangan menyimpan atau menampilkan token, password, atau secret.
- Jangan menjalankan perintah yang berpotensi merusak sistem tanpa konfirmasi.
